# Turbine widgets
Small turbine widgets that tell you the status behind the ECU codes in plain text.

## Turbine widget
Jakadofsky, evoJet, PBS, HORNET, JetCat, KingTech, AMT, Xicoy/Kolibri, JetCentral, Kolibri NG, Swiwin, Linton

## Linton widget
Only works with the Linton turbines.

## Files
### turbine.lua & linton.lua 
Uncompiled code, human readable can be run on the transmitter but takes more memory space.

### turbine.lc & linton.lc 
Compiled code, takes less space in the transmitters memory.
